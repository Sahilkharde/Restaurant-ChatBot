Q: Can I set up an account for the signer and create the notarization request?
A: Yes, you can create and manage a notarization request from your account, even if the signer is different from the account holder. However, please note the following:
The account name should match the signer’s name to ensure compliance and smooth verification.
The signer must complete their own ID Verification and Knowledge-Based Authentication (KBA) process.
Selfie verification is required as part of the ID process, and it must be completed by the signer themselves.
During the notarization session, the signer must appear live with the notary via audio-video connection (not the account holder).
The account holder can upload documents, pay fees, and manage scheduling, but the notarization must be performed by the signer.
For multiple signers, each signer must complete their own ID verification and join the session individually.
These requirements ensure your notarization is valid and compliant with state RON laws.
